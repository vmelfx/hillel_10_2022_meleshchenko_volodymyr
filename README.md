## Quick start
 
### Install deps
```bash
# Instal pipenv
pip install pipenv

# Activate virtual env
pipenv shell

# Install deps
pipenv sync
```

#### Additional
```bash
# Regenerate Pipfile.lock
pipenv lock

# Install deps from Pipfile.lock file
pipenv sync

# Pipenv lock & pipenv sync
pipenv update
```
